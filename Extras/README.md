# Extras

Aqui poderemos colocar qualquer coisa extra, que não esteja diretamente ligada a aula dada porém ainda faça parte do assunto, como projetos pessoais que alguém queira compartilhar para que outros analisem o código e aprendam algo (ou ajudem a melhorar se possível).

Pra manter uma ordem que seja fácil de acessar, vamos colocar cada projeto em uma pasta própria, nomeada por projeto, independente de ser um projeto único ou um conjunto de scripts relacionados diretamente ou indiretamente ao projeto em questão.
