Version 0.0.1 - Pre-Alpha

Scripts:

-> InterfaceController.py (Gerencia Interface)
-> MainController.py (Gerencia os Dados do Jogo)
-> RandomCalculator.py (Gerencia a Mecânica do Jogo/Cálculos/Loops)

Adicionado:

-> Escolha de dificuldade (fácil, médio e difícil);
-> Escolha de tipo de cálculo (adição e subtração);
-> Cálculos com variações de números dependendo da dificuldade;
-> Números randômicos sendo gerados para exibir sempre contas aleatórias assim que uma é respondida;
-> Sistema de loop (conta respondida -> gera novos números -> calcula os números -> exibe os números -> aguarda o input para verificar (F) ou (T));
-> Sistema de abandonar o jogo e voltar pra escolha de dificuldade e tipo de conta (escrevendo "exit" ou "sair" em vez de digitar um número ocasiona o jogo a voltar para o menu de escolha).

Pretendidos:

-> Exceção para que se o jogador escrever outra letra além dos nomes pretendidos de scape, o jogo continue mostrando a conta até que ela seja respondida;
-> Sistema de pontuação (+1 ponto pra cada resposta correta);
-> Sistema de vidas (+3 vidas [cada resposta errada desconta uma vida]);
-> Multiplicação e divisão;
-> Equações do primeiro grau;
-> Randomicidade de tipos de cálculos (podendo vir adição, subtração, divisão...no mesmo jogo);
-> Armazenamento de dados (pontuação máxima).

Futuras:
